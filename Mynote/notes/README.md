# 笔记文件夹

本文件夹用于存放所有Markdown格式的笔记文件。

## 使用说明

1. 在本文件夹中创建以.md结尾的Markdown文件
2. 每个文件代表一篇笔记
3. 提交并推送更改到GitHub仓库
4. 网站会自动加载并显示这些笔记

## 笔记格式

请使用标准的Markdown格式编写笔记：

```markdown
# 笔记标题

## 二级标题

### 三级标题

正文内容，可以包含：
- 列表项1
- 列表项2
- 列表项3

**粗体文本** 和 *斜体文本*

[链接文本](https://example.com)

```

## 提示

- 文件名建议使用有意义的英文或拼音，避免特殊字符
- 可以使用相对路径引用其他笔记或资源
- 定期备份你的笔记内容
